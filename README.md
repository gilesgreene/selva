# VedaSales — Ecommerce Intelligence

Find winning products before they peak. VedaSales aggregates data from Reddit, Etsy, YouTube, and Google Trends to surface trending products for ecommerce sellers.

## 🚀 Features
- **Trend Dashboard**: Real-time trending products feed with internal "Opportunity Score".
- **Multi-Source Intelligence**: Data from Reddit (sentiment), Etsy (sales/listings), YouTube (reviews), and Google Trends.
- **AI-Powered Insights**: Concise product summaries generated by Claude 3.5 Sonnet.
- **Keyword Explorer**: Deep dive into any niche or keyword interest over time.
- **Watchlist**: Track potential winners and get trend alerts.

## 🛠 Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v4
- **Database**: Supabase (Postgres)
- **Auth**: Clerk
- **Payments**: Stripe
- **AI**: Anthropic SDK

## 📦 Setup Instructions

### 1. Environment Variables
Copy `.env.example` to `.env.local` and fill in your keys:
```bash
cp .env.example .env.local
```

### 2. Database Setup
1. Create a new Supabase project.
2. Run the SQL schema found in `artifacts/schema.sql` (or `implementation_plan.md`) in the Supabase SQL Editor.

### 3. Install Dependencies
```bash
npm install
```

### 4. Run Development Server
```bash
npm run dev
```

### 5. Seeding Data
Initial seed data can be populated by creating an API route or running a script that calls the aggregation services for the top 50 keywords.

## 📜 Database Schema
- `users`: User profiles and subscription plans.
- `products`: Core product trend data.
- `trend_history`: Historical trend scores for charting.
- `reddit_posts`, `etsy_listings`, `youtube_videos`: Raw source data.
- `watchlist`: User-saved products.

## 🤖 AI Summary Prompt
VedaSales uses Anthropic's Claude 3.5 Sonnet to analyze aggregated signals and provide a "3-4 sentence opportunity insight" for every product.
